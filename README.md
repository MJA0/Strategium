# Strategium

**Strategium** is a **Warhammer 40,000 roster builder and tabletop companion** designed to help you manage your army lists and reference rules while playing.

**Try it here:**  
https://mja0.github.io/Strategium/
https://strategium.netlify.app

---

## Features

**Roster Builder**
- Build and manage Warhammer 40k army lists
- Customize units. Size, wargear, and enhancements

**Play Mode**
- Quickly view your army’s **datasheets**
- Reference **weapons/units abilities and keywords**
- Access **stratagems and rules** during gameplay

**Mobile Friendly**
- Installable as a **Progressive Web App (PWA)**
- Phones and tablet compatible
- Designed for quick reference during tabletop matches
